**[Slide 4: Core Code - Histories Router]**
- **Topic:** Implementation of History Lookup
- **Code Snippet:**
```javascript
router.get('/:productId', (req, res) => {
    const blockchain = req.app.get('blockchain');
    const { productId } = req.params;
    
    // Encapsulate business logic using helper methods
    const transactionHistory = blockchain.getTransactionsByProductId(productId);

    if (transactionHistory.length === 0) {
        return res.status(404).json({ error: `History not found for product ID [${productId}].` });
    }

    res.json({
        result: "Success",
        history: transactionHistory
    });
});
```
- **Key Logic:** Uses `getTransactionsByProductId` helper to traverse the chain and extract relevant transactions.

**[Slide 6: Core Code - Mine Router]**
- **Topic:** Proof of Work & Block Creation
- **Code Snippet:**
```javascript
router.post('/', async (req, res) => {
    const { minerAddress } = req.body;
    // ... validation ...
    
    // Perform Proof of Work (PoW)
    const nonce = await blockchain.proofOfWork(previousBlockHash, currentBlockData);
    const blockHash = blockchain.hashBlock(previousBlockHash, currentBlockData, nonce);

    // Create new block
    const newBlock = blockchain.createNewBlock(nonce, previousBlockHash, blockHash);

    // Reward transaction is added to the NEXT block
    blockchain.createNewTransaction('00-REWARD-SYSTEM', minerAddress, 'MINING-REWARD');

    res.json({ result: "Success", block: newBlock });
});
```
- **Key Logic:** Calculates valid `nonce` and `hash` (PoW), adds block to chain, and queues a reward transaction for the next block.

**[Slide 8: Core Code - Consensus Algorithm]**
- **Topic:** Consensus (Longest Chain Rule) Implementation
- **Code Snippet:**
```javascript
router.get('/consensus', async (req, res) => {
    // ... request blockchain data from all nodes ...
    const blockchains = await Promise.all(requestPromises);
    
    // Find the longest valid chain
    for (const response of blockchains) {
        const remoteChain = response.data.chain;
        if (remoteChain.length > maxChainLength && blockchain.chainIsValid(remoteChain)) {
            maxChainLength = remoteChain.length;
            newLongestChain = remoteChain;
        }
    }

    // Replace chain if a longer valid one exists
    if (newLongestChain) {
        blockchain.chain = newLongestChain;
        res.json({ message: 'Chain replaced with the longest valid chain.' });
    }
});
```
- **Key Logic:** Synchronizes data by adopting the longest valid chain from the network.
